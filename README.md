# Web Mini Projects

A collection of small web development projects built while learning HTML, CSS, and JavaScript. Each project focuses on practicing core front-end concepts by creating interactive and responsive web applications.

## Projects

- Personal Portfolio
- Login Page
- Counter App
- Number Guessing Game
- Rock Paper Scissors
- Dice Roller
- Temperature Converter
- Digital Clock
- Stopwatch
- Calculator
- Weather App

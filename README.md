# Web Mini Projects

A collection of small web development projects built while learning HTML, CSS, and JavaScript. Each project focuses on practicing core front-end concepts by creating interactive and responsive web applications.

## Projects

- Personal Portfolio
- Login Page
- Counter App
- Number Guessing Game
- Dice Roller
- Temperature Converter

## Concepts Practiced

- HTML5 Structure
- Semantic HTML
- CSS Styling
- Flexbox
- Responsive Layouts
- DOM Manipulation
- Event Handling
- Variables & Data Types
- Arithmetic Operators
- Conditional Statements
- Loops
- Functions
- Arrays
- Random Number Generation
- Dynamic Content Rendering

## Goal

This repository documents my journey of learning front-end web development through hands-on projects rather than only following tutorials. Each project helps strengthen my understanding of HTML, CSS, JavaScript, and the fundamentals of building interactive web applications.
